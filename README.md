## Install the Polymer-CLI

First, make sure you have the [Polymer CLI](https://www.npmjs.com/package/polymer-cli) installed. 

## Install the bower dependencies with polymer using...

```
$ polymer install
```

## Viewing Your Application

```
$ polymer serve --open
```
